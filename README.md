# 🍽️ Restaurant Revenue Prediction

## Title : Restaurant-Revenue-Prediction-ML-Project
Restaurant Revenue Prediction using Machine Learning with EDA, Linear Regression, and Random Forest to analyze key factors influencing restaurant revenue.

#### 📌 Project Overview :

The Restaurant Revenue Prediction project focuses on analyzing restaurant operational data and building machine learning models to predict the revenue generated by restaurants.

The project combines data cleaning, exploratory data analysis (EDA), visualization, and predictive modeling to understand the key factors influencing restaurant revenue.

The primary objective is to identify the relationship between operational variables such as order volume, menu size, and restaurant categories and use these insights to predict revenue using regression models.

This project demonstrates an end-to-end data science workflow, including:

- Data loading and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization and statistical insights
- Machine learning model building
- Model evaluation and comparison


#### 1. 📂 Dataset :

The dataset contains restaurant-level operational information, including :

- Id : Unique Identifier for each restaurant
- Name : Name of the Restaurant
- Category : Type of Restaurant(Burge, BBQ, etc.)
- Franchise : Whether the restaurant is a franchise.
- City : Location of the Restaurant
- No. of Items : Number of Items Offered.
- Order Placed : Number of Orders Received
- Revenue - Total Revenue Generated(Target Variable)

Cities Included in the Dataset :

- Bengaluru
- Mumbai
- Noida
- Pune
- Gurugram

#### 2. Data Cleaning & Preprocessing : 

Before performing analysis, several preprocessing steps were carried out to ensure the dataset was clean and suitable for modeling.

**Steps Performed :**

- Checked dataset shape and column information
- Verified data types of each column
- Checked for missing/null values
- Checked for duplicate records

**Observations :**

#### 4. 🤖 Machine Learning Models :
Two Regression models were developed to predict restaurant revenue.

**Simple Linear Regression (Baseline Model) :**
A baseline model was created using:

- Independent Variable: Order Placed
- Target Variable: Revenue

*Model Performance :*

| Metric   | Value           |
| -------- | --------------- |
| R² Score | ~0.91           |
| MAE      | 318,430         |
| MSE      | 186,768,572,957 |
| RMSE     | 432,167         |

The high R² score (~0.91) indicates that order volume explains a large portion of the variation in revenue.

*Model Diagnostics :*

- Created comparison dataframe of Actual vs Predicted Revenue
- Calculated prediction error
- Performed residual analysis
- Visualized Actual vs Predicted values using line plot

#### Residual Error Using Line Plot
![Linear Regression Residual Error](https://github.com/aishwaryasavanth22/Restaurant-Revenue-Prediction-ML-Project/blob/bf3337dfa1cf142e011346d5c1fcee6c084c361e/Visualizations/Linear%20Regression%20Residual%20Plot.png)

Insights : After Plotting the residual for actual and predicted value, we see that errors are centered around 0 this indicates our model is good.

#### Comparing Actual Revenue Vs Predicted Revenue
![Simple Linear Regression Model Plot](https://github.com/aishwaryasavanth22/Restaurant-Revenue-Prediction-ML-Project/blob/bf3337dfa1cf142e011346d5c1fcee6c084c361e/Visualizations/Linear%20Regression%20Actual%20Revenue%20Vs%20Predicted%20Revenue%20Line%20Plot.png)

**Random Forest Regressor :**

To explore a more advanced approach, a Random Forest Regressor was implemented.

Model Configuration :

- n_estimators = 100
- random_state = 42

*Model Performance :*

| Metric   | Value           |
| -------- | --------------- |
| R² Score | ~0.84           |
| MAE      | 364,108         |
| MSE      | 228,884,537,147 |
| RMSE     | 478,418         |

Although Random Forest is a powerful algorithm, the Linear Regression model performed slightly better, indicating that the relationship between orders and revenue is largely linear.

![RF Model Actual Vs Predicted Line Plot](https://github.com/aishwaryasavanth22/Restaurant-Revenue-Prediction-ML-Project/blob/bf3337dfa1cf142e011346d5c1fcee6c084c361e/Visualizations/Random%20Forest%20Regressor%20Actual%20Revenue%20Vs%20Predicted%20Revenue%20Comparison%20Using%20Line%20Plot.png)

#### 5. 📊 Business Recommendations :

Based on the analysis :

**1. Focus on Increasing Order Volume :** 

The analysis revealed a strong positive relationship between the number of orders placed and revenue generated. Restaurants receiving a higher number of orders tend to generate significantly more revenue.

Restaurant businesses should focus on strategies that increase order volume, such as:

- Online food delivery partnerships
- Promotional campaigns and discounts
- Loyalty programs and customer retention strategies
- Improving service efficiency to handle higher order volumes

Increasing order volume is likely to have the most direct impact on revenue growth.

**2. Expand Menu Variety Strategically :** 

The analysis also showed a positive relationship between the number of menu items and restaurant revenue. Restaurants offering a broader selection of menu items tend to attract more customers.

Restaurants should consider expanding their menu offerings while ensuring:

- Menu items align with customer preferences
- Kitchen operations remain efficient
- Quality and consistency of food are maintained

A well-balanced menu variety can attract a wider customer base and increase sales opportunities.

**3. Use Predictive Models for Revenue Forecasting :** 

The machine learning models developed in this project can help restaurants predict future revenue based on operational factors such as order volume and menu size.

- Restaurant management can use predictive analytics to: 
    - Forecast revenue trends
    - Plan staffing and inventory requirements
    - Optimize marketing campaigns
    - Support data-driven business decisions

Using predictive insights can help businesses reduce uncertainty and improve financial planning.
